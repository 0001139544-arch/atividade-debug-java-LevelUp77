# Atividade Debug Java

Este projeto tem como objetivo criar um programa em Java que solicita o nome de um aluno e duas notas.

O sistema calcula a média do aluno e informa sua situação final:

- Aprovado (média maior ou igual a 7)
- Recuperação (média entre 5 e 6.9)
- Reprovado (média menor que 5)

O programa utiliza entrada de dados pelo usuário via terminal.
